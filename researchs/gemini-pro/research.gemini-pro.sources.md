# Sources for gemini-pro
# Kaynakça ve Referanslar

Bu proje geliştirilirken aşağıdaki kaynaklardan yararlanılmıştır:

1. **Google Gemini (LLM):** SQL Injection zafiyetlerinin tespiti, analizi ve güvenli kod örnekleri (Prepared Statements) hakkında prompt mühendisliği ile bilgi toplanmıştır.
2. **OWASP (Open Web Application Security Project):** Yaygın web zafiyetleri ve korunma yöntemleri.
3. **PHP.net Resmi Dokümantasyonu:** `mysqli` kütüphanesi ve parametreli sorgu kullanımı.
